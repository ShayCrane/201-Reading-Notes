## Virtualizing a Router with pfSense

### Introduction to pfSense- An Open Source Firewall and Router Platform
*https://turbofuture.com/computers/Introduction-to-pfSense-An-Open-Source-Firewall-and-Router-Platform*

*(This article includes a complete FAQ, "Questions & Answers" at the bottom of the article, that is worth pulling up when using pfSense.)*



*What is pfSense?*
	- customized distro of FreeBSD (name of another open source OS)
	- turns old computer into router and firewall
		- for home or large corp. network

	- More uses for pfSense:
		- LAN/WAN Router
	- Internet Cafes
	- Wireless Hotspot (Captive Portal)
	- VPN Router
	- Firewall
	- DHCP / DNS Server
	- Wireless Access Point
	- Transparent Squid Proxy Server
	- Multi Wan Router or Load Balancer
	- DNS Blacklist
	- Port Forwarding / NAT (Network Address Translation)

*Why pfSense over another router?*
	- for features not offered on your current router
	- customizability, flexibility
	- saves you the cost of buying a router
	- can embed in smaller appliances for better aesthetics
		- there is a specific embeded version for download

*Using pfSense*
	- set up on old computer
		- must have at least 2 network cards installed
		- full installation instructions linked to in article: *http://samkear.com/how-to-guides/setting-up-a-pfsense-router*

*Support*
	- free forums, mailing list, IRC
	- BSD offers commercial support for enterprise customers
		- subscribe for five hours of support a year: 
			- access to auto configuration backup utility; and
			- and pfSense book online.
